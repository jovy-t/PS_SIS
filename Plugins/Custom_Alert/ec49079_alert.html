## This is a custom alert that can be added to the student profile page. It uses the same classes as the default alerts. ## Filepath: /admin/alerts/user/ec49079_alert.html

~(studentname)

Education Code 49079 Notification

This student has, within the past three years, engaged in or is reasonably suspected of engaging in conduct requiring disclosure under California Education Code 49079.

This information is confidential. Please contact site administration with questions.
